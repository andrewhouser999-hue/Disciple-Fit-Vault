## 1. Model Foundation
*The documents that define what Covenant Identity Coaching is. These are not phase-specific — they are the conceptual and theological ground the entire system stands on.*

---

**✅ Disciple Fit Covenant Identity Coaching — White Paper.md**
High-level practitioner-facing overview of the Covenant Identity Coaching model. States the core claim and distinguishes the model from adjacent approaches. The compressed version of the model's thesis — read before the Full Model Reference to orient the high-altitude frame before going deep.

**✅ What Is Covenant Identity Coaching — Client-Facing.md**
Plain-language explanation of the model for prospective clients. No jargon. Describes the problem being addressed, what the coaching does, what to expect, and who it is for. The first document a prospective client sees.

**✅ Developing a Biblical Understanding of Covenantal Relationship with God.md**
Biblical-theological research document grounding the model's core claim. Surveys covenant relationship with God across Scripture (creation covenant through new covenant) and establishes it as the constitutive foundation of covenant identity work. The exegetical basis for why covenant relationship is the governing frame, not a thematic overlay on an independent anthropology. Informs grounded declaration language and prevents the nine identity layers from functioning as slogans rather than rooted truth.

**✅ Covenant Identity Coaching — Full Model Reference.md**
Comprehensive 16-section practitioner reference. All frameworks defined with jargon explained inline. The coach's complete model document — theological architecture, psychological integration, how the model differs from therapy and generic coaching. The canonical source for all model language.

**✅ Biblically Integrated Covenant Identity Coaching — Framework.md**
*(Located in Coaching Business folder)*
Original 14-section framework document. The conceptual origin of the model. Predates the Full Model Reference. Overlap is intentional — this is the development record.

**🔲 Biblical Foundation Reference — The 10 Layers**
Each of the ten covenant identity layers mapped to their specific biblical texts with exegetical depth. Not a devotional — a practitioner reference giving the coach textual and theological substance behind each layer. Supports grounded declaration work and prevents the layers from functioning as slogans rather than rooted truth.

## 4. Intake System
*Completed before and around session 1. Establishes the documented baseline against which all growth is measured.*

---

**✅ Covenant Identity — Two-Tier Intake Protocol.md**
Overview document describing the two-tier structure of the Covenant Identity intake process. Tier 1: the written Tally form completed before session 1. Tier 2: verbal follow-up conducted within session 1. Explains the protocol logic, what each tier captures and why, and how they work together to establish the diagnostic baseline. Read before the individual intake documents (Design Basis, Annotation Reference, Intake Addendum, Build Guide).

**✅ Covenant Identity Intake — Design Basis & Annotation.md**
Medium-agnostic master reference for the intake form. Covers the form's primary job, design principles, tensions and resolutions, diagnostic target map, complete question set with full coach annotations, session 1 integration notes, and medium adaptation notes. The permanent rationale document — use this when adapting the form to a new medium.

**✅ Covenant Identity Intake — Tally Build Guide.md**
Step-by-step implementation guide for building the intake form in Tally.so. Exact question text, question types, section structure, welcome/thank you page copy, Airtable integration settings, and in-session verbal follow-up reference table.

**✅ Covenant Identity Intake — Coach Annotation Reference.md**
Standalone quick-reference for reviewing a completed intake form before session 1. Clean per-question format: diagnostic target, how to read answers, follow-up questions to prepare, referral watch signals. Working hypothesis template at the end.

**✅ Covenant Identity — Intake Addendum (Coach).md**
Coach-completed document (not client-facing). Part A: 12 verbatim language fields pulled from the Tally form. Part B: 4 supplemental fields gathered verbally in session 1 — felt-sense description, present-story statement, covenant identity self-assessment, coach synthesis statement. Summary Baseline table. The foundational "before" record for the entire engagement.

**✅ Covenant Identity — Pre-Discovery Screen — Tally Build Guide.md**
Step-by-step implementation guide for building the pre-discovery call screening form in Tally.so. A shorter screen designed to run before the discovery call, not a substitute for the full intake. Screens for basic fit indicators (faith context, presenting concern, relevant history flags) without the full diagnostic depth of the main intake form. Allows the coach to enter the discovery call with basic context already in hand.

*Background reference documents and standalone tools not specific to a single phase.*

**✅ Covenant Identity — Practitioner Reference Index.md**
Navigation hub for the diagnostic/theoretical cluster. Theoretical justification for using Thompson, Allender, Johnson, and Collins together within a biblically governed frame. Governing position: Christian psychology (not classic integration or secular synthesis). Two integration criteria (confirmatory vs. additive). Per-contributor justification: what each contributes, the limits on what is imported, and what is explicitly excluded — Siegel's telos, Erikson's normative identity framework, CBT as sufficient for change, humanistic telos. **✅ Covenant Identity — Whole-Person Identity Framework.md**
Survey-depth practitioner anthropology reference. Governing framework: imago Dei (origin, distortion, telos), soul's faculties (mind, will, affections), and five-dimension whole-person structure (cognitive, affective, volitional, relational, somatic). Mechanism layer (Thompson — neurological/relational formation), diagnostic layer (Allender — character wound; Collins — clinical boundary). Scope boundary and sources/boundary notes. Scripture as governing plumbline. The conceptual frame for what the coach is looking at in every session. Contributors: Thompson, Allender, Johnson, Collins.

**✅ Covenant Identity — Functional vs. Confessional Theology — 20 Diagnostic Contrasts.md**
Practitioner reference presenting 20 paired contrasts between functional belief (operationally evident from behavior, patterns, and affect) and confessional belief (stated theology). Supports precise lie identification and prevents coaching from stopping at theological restatement of truths the client already acknowledges.

**✅ Covenant Identity — Diagnostic Lens Transition Logic.md**
The spine of the entire diagnostic sequence. Includes the two-branch Stage 4 split: cognitive distortion barrier → Stage 4a; implicit memory barrier → Stage 4b. Read before all individual diagnostic tools — this document governs the "when" that the individual tools address the "how."

**✅ Covenant Identity — God-Exclusion Map.md**
Standalone practitioner reference for Layer 4 of the Affections Discovery protocol. Eight functional God-exclusion domains (performance, control, relationships, comfort, image, achievement, safety, autonomy) with in-session question bank, diagnostic signals, and transition logic to Layer 5 and Stage 2. Grounded in the Jeremiah 2:13 / Isaiah 31:1 framework: God as the spring of living water functionally displaced by other sources of identity, security, or comfort.

**✅ Five Intervention Modalities — Definitions & Deep Elaboration.md**
Practitioner reference defining and elaborating the five core intervention modalities available within the model: Declaration, Lament, Narrative Re-Authoring, Behavioral Experiment, and Somatic Integration. Governs how the coach selects and sequences interventions across the four phases rather than defaulting to a single approach.

**✅ Covenant Identity — Character Wound Diagnostic Tool.md**

#### Sanctification Module — The Believer's Part

**Deployment:** Middle sessions — after Affections Discovery has surfaced a false covenant object without defensive collapse. Not at intake. Not during acute crisis.
**Target client:** Moderate theological literacy with low integration; high ACE prevalence; presenting with the gap between knowing and experiencing.

**✅ Sanctification — Teacher's Manual.md** *(Coach)*

Four failure modes (Warrior/Hermit/False Noble/Mode 4 — no inner access as an ACE-driven overlay), biblical framework (asymmetric synergism, definitive/progressive sanctification distinction, Phil 2:12-13 as the organizing hinge), soil type mapping (Rocky/Thorny), ACE-sequenced formation tiers (Tier 1–3), scope boundary, diagnostic tool connections, and failure mode guards.

**✅ Sanctification — Coach Diagnostic.md** *(Coach)*
Eight-step pre/post-session diagnostic: deployment readiness checklist, literacy-vs-integration check (three response types), soil type reading, Mode 4 screen (with Mode 2 vs. Mode 4 distinction table), failure mode identification (observable indicators and diagnostic questions per mode, multiple failure mode routing table), wound type filter, formation track assignment table, and scope boundary quick reference. Routing card used at end of session — not during.

**✅ Sanctification — Session Facilitation Guide.md** *(Coach)*
Scripted language for Sessions 1–2: opening, literacy check (three response types), framework introduction (Phil 2:12-13 paraphrased), failure mode naming, practice assignment (Tracks A–C). Session 2: practice debrief, definitive/progressive distinction, soil type mapping, track advancement, closing. Resistance handling by all four failure modes. Return session protocol (Session 3+). Transition protocol: readiness signals, routing table by diagnostic signal, handoff language to Affections Discovery Stage 2, Secured in Father, and Lie Eliminator.

**✅ Sanctification — Client Study.md** *(Client)*
2–3 page pastoral-warm study: connects Phil 2:12-13 via the *gar* ("because"), explains the order of the players (God as primary agent, Spirit as means, believer as real but empowered agent), disciplines as conditions not causes, definitive vs. progressive distinction, names the knowing-feeling gap as non-defect. Phil 2:12 paraphrase replaces "fear and trembling" with italicized meaning throughout, with footnote explaining the change for ACE-prevalent populations.

**✅ Sanctification — Practice Reference Card.md** *(Client)*
Track A (Phil 2:13 Meditation — Mode 1/Warrior), Track B (Rom 6:11 Reckoning — Mode 2/Hermit and Mode 4), Track C (Immunity to Change Map — Mode 3/False Noble); coach marks assigned track before distributing. Page 1: Tracks A and B with practice instructions and faithful vs. going-through-the-motions guidance. Page 2: Track C — full four-column ITC client worksheet with column-by-column prompts.

**✅ Sanctification — ITC Practitioner Reference.md** *(Coach)*
Full Immunity to Change standalone practitioner reference: four-column protocol with facilitation guidance per column, reading what the client brings back (Column 3 content patterns table, Column 4 assumption category patterns), routing into the Covenant Identity framework (Column 4 → Affections Discovery Layer 5, Character Wound Diagnostic, Lie Eliminator, Secured in Father, Diagnostic Lens Stage 2), in-session facilitation guide for Columns 1–2, return session protocol for Columns 3–4, common obstacles, scope boundary notes, and cross-references. Companion to Track C of the Practice Reference Card.
