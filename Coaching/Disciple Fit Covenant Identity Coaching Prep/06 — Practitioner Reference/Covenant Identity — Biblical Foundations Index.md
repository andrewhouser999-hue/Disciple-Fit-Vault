# Covenant Identity — Biblical Foundations Index

*Practitioner reference — what Scripture each architectural/methodological decision in the model rests on, and where the full case is made.*

---

## Purpose

This index does not perform exegesis. It maps each major **structural or methodological decision** in the Covenant Identity model — not individual lie/identity verse assignments, not contributor-to-element mappings, not the per-layer promise catalog (those are handled by [[Covenant Identity — Author-Framework Mapping]], [[Covenant Identity — New Covenant Promises — Comprehensive Reference]], and the tools themselves) — to:

1. its primary scripture text(s),
2. what those texts establish *for that decision*,
3. the document containing the full exegetical case, and
4. a status flag.

Use it when a design decision is questioned or extended, or when a new practitioner asks "why is this built this way."

**Status key:** `Developed` (full exegetical case exists) · `Assumed` (cited, not exegeted) · `Contested` (real interpretive fault line — see footnote) · `Gap` (no documented grounding — named, not filled).

**Relationship to Coaching Preparation's Biblical Frameworks tier system:** that folder's A/B/C tier system operates one level upstream, at the practice level. [[Affections Discovery Process — Affections, Trust & Priorities]] (Section 2, Row 3 below) is the one place a CI decision draws directly on it. This index does not restate that tier system.

> ⁵ **Addendum** — Eph 4:22-24's "put off"/"put on" are aorist (definitive, decisive); "being renewed" is present tense (ongoing). Mainstream commentary (O'Brien, Moo, Wright, Lincoln) reads this as both/and. This tense distinction is not yet discussed in the Author-Framework Mapping's structural-framework section — adding it would tie this row directly to Section 1, Row 2 (definitive/progressive sanctification) via the same grammatical pattern in a second text.

> ⁶ **Addendum** — Rom 6:11's *logizesthe* ("reckon/consider") links to the *logizomai* reckoning-language of Rom 4:3-22 (justification). Candidate as a second master verb for "declare what is already true," alongside Gal 3:27. **Flag:** Rom 6:11 is frequently mis-cited as license for self-talk/positive-thinking; its actual force is "you are dead to sin — stop transacting with it," not "speak it until true." Not yet in any document — would need careful framing if added.

> ⁷ **Contested** — 2 Cor 10:4-5's original referent is Paul's apostolic authority against rival teachers in Corinth (Harris, Barnett, Thrall, Garland), not personal cognitive lies. The Lie Eliminator's use is an analogical extension: the *structure* (arguments/strongholds set against true knowledge) transfers; the *referent* does not. Not currently flagged in the Lie Eliminator document itself.

> ⁸ **New candidates** (2026-06-11 /bible research, not yet in any document) — **Isaiah 28:14-18** ("you have made a covenant with death... we have made lies our refuge... your covenant with death shall be annulled") as a parallel for the Agreement Audit (locating/renouncing the moment of agreement with a lie); same analogical-extension caveat as ⁷ (original referent is Judah's political alliance with Egypt against Assyria). **James 1:22-24** (*paralogizomenoi*) as a stronger, less-contested anchor for the same "belief without formational landing" gap — see Section 2, footnote 3, where this same text fills a *different* documented gap. One text, two open gaps.

> ⁹ **Gap** — candidate texts exist (Deut 6:4-9, Deut 8:11-18, Ps 42, Ps 103, Lam 3:21-24, all from 2026-06-11 /bible research) but no document develops the case for *why* spoken self-declaration is itself a biblical pattern, as distinct from *hagah*'s meditation/rehearsal (Row 3 above, already documented). May warrant a small standalone reference document — see "Open Items."
