# Covenant Identity — Tool Library
*Practitioner Reference — complete inventory of in-session and between-session tools*

This document inventories all tools available within the Covenant Identity Coaching model, organized by deployment context and depth layer. It is a navigation document; each entry points to the reference document where the tool is fully described.

**Depth layer notation:**
- **[Explicit]** — operates at the cognitive/propositional level; accessible to direct verbal engagement
- **[Implicit]** — operates below the cognitive level: somatic, relational, imaginative, or formation-based; not fully accessible through reasoning alone
- **[Mixed]** — engages both layers

**Coach-safe scope:** All tools listed here are deployed within coaching scope. Where a tool approaches the clinical ceiling, a scope note is included.

**Related:** [[Covenant Identity — Practitioner Competency Framework]] — for the competency domains required to deploy these tools, particularly implicit-layer tools.

## Tool Deployment Principles

**1. Diagnosis precedes assignment.**
No tool is assigned before the diagnostic picture is specific enough to match it. The wrong tool applied to an undiagnosed presentation wastes formation energy and produces discouragement.

**2. Explicit tools cannot reach implicit barriers.**
When Stage 4b is operating — the client knows the truth but cannot feel it — more declaration work, more CBT, more content will not move the barrier. The implicit layer requires implicit engagement: relational presence, somatic readiness, encounter-based practice, accumulated relational experience.

**3. Between-session tools require debrief.**
The client's response to a tool in practice is diagnostic data. The declaration that produces tears in session 1, flatness in session 2, and nothing by session 4 is telling the coach something about the schema operating beneath it.

**4. Contraindications are absolute at the coaching ceiling.**
Secured in Father and Stage 4b work both have clinical referral boundaries. Unprocessed trauma, active dysregulation, or dissociative features are not a signal to try harder — they are referral signals.

**5. The coach's implicit competencies are the container for all explicit tools.**
Declaration work, CBT tools, and schema interruption all operate inside the relational container the coach creates. A technically correct explicit intervention delivered from anxious, distracted, or defended presence will not reach depth. See [[Covenant Identity — Practitioner Competency Framework]] Domain 3.
