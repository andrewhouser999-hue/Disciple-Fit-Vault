# Covenant Identity — Practitioner Reference Index
*Entry point for the Covenant Identity coaching model — practitioner reference cluster*

## The Cluster

This index covers five documents that form the diagnostic and theoretical core of the Covenant Identity coaching model. Three are practitioner references (this folder); one is an in-session tool (04 — Session Tools); one is upstream source material.

---

### Start Here — If You're New to the Model

> **[[Covenant Identity — Integration Rationale]]** → **[[Covenant Identity — Whole-Person Identity Framework]]** → **[[Covenant Identity — Diagnostic Lens Transition Logic]]**

Read them in that order once. After that, use the sequence and tools below as in-session references.
