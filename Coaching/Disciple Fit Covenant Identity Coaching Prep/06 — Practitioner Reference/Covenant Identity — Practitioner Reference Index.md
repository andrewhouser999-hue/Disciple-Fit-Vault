# Covenant Identity — Practitioner Reference Index
*Entry point for the Covenant Identity coaching model — practitioner reference cluster*

---

## The Cluster

This index covers the documents that form the diagnostic and theoretical core of the Covenant Identity coaching model. Documents span practitioner reference (this folder), in-session tools (04 — Session Tools), and upstream source material.

---

### Start Here — If You're New to the Model

> **[[Covenant Identity — Integration Rationale]]** → **[[Covenant Identity — Whole-Person Identity Framework]]** → **[[Covenant Identity — Diagnostic Lens Transition Logic]]** → **[[Covenant Identity — Practitioner Competency Framework]]** → **[[Covenant Identity — Stage 4b Implicit Level Practitioner Reference]]** → **[[Covenant Identity — Tool Library]]**

Read them in that order once before your first engagement. The first three establish the theoretical and diagnostic foundation. The Competency Framework translates the model into what you must be able to do. The Stage 4b reference gives you the deep skill reference for the implicit-level competency domain. The Tool Library gives you the full inventory of what is available.

## What Is Not in This Cluster

- **Session scripts** — this cluster is diagnostic frame and practitioner reference, not scripted session content
- **Client-facing materials** — all documents here are coach-facing only, with the exception of Formation Framework Modules (see above), which contain both coach-facing and client-facing documents within their subfolders
- **Covenant Truth content** — Stage 3 tools (declaration, Covenant Promise Map, Identity Installation) are separate session tools in `04 — Session Tools`
- **CBT technique reference** — Stage 4a tools are referenced in the Tool Library and the Transition Logic, but the technique reference itself is assumed practitioner knowledge; not documented here. Stage 4b, by contrast, is documented — see [[Covenant Identity — Stage 4b Implicit Level Practitioner Reference]]
- **Between-session tool documents** — the Tool Library inventories between-session tools, but the actual documents (*Secured in Father*, Lie Eliminator, Identity Installer, worksheets) are located in `Coaching Me/` and `08 — Between-Session Materials`
- **Model Foundation documents** — the Practice Definition, White Paper, Manifesto, Theory of Change, The CIC Promise, and related documents in `01 — Model Foundation` establish what the practice is; this cluster assumes that foundation and moves straight to diagnostic and theoretical mechanics
