# Covenant Identity — Between-Session Library — Design Rationale

*Orientation and design rationale for the full between-session tool library. One section per tool: purpose, priority verdict, pre-build design decisions, key distinctives, and system connections. Does not duplicate tool content — open the corresponding file for the full tool.*

---

## Overview

This document exists for three purposes:

1. **Re-entry orientation** — when returning to this library after time away, this is the first document to read. It holds the rationale behind each tool, not the tool content.
2. **Future iteration** — when a tool needs updating, the design decisions here record what the original constraints were and what was being protected against.
3. **Deployment discernment** — when deciding which tool to assign in session, the precondition logic and "connects to" notes here are faster than opening each individual file.

**Evaluation prompt:** [[AI Prompt — Between-Session Tool Evaluation]] — eight criteria for vetting any new between-session tool before building.

**Full tool inventory (in-session + between-session):** [[Covenant Identity — Tool Library]] — for deployment context, depth layer classification, and how between-session tools fit within the broader model tool architecture.

**Full file index:** [[Covenant Identity — System Document Master List]] and the memory file `covenant_identity_file_index.md`.

## Evaluation Prompt — Eight Criteria Summary

*Full prompt in [[AI Prompt — Between-Session Tool Evaluation]]. Criteria in brief:*

1. **Mechanism Fit** — does the tool engage implicit memory / affections / beholding, or only the cognitive layer?
2. **Phase Alignment** — which phase(s) does it serve; what is the deployment risk?
3. **Character Wound Differentiation** — genuinely adapted per type, or universal?
4. **God-Rep Sensitivity** — does it require calibration by God-rep type, and how?
5. **Client Readiness Floor** — minimum formation readiness required to help rather than harm
6. **Practitioner Load** — coach-assigned once / coach-customized per client / client-autonomous
7. **Spirit-Dependency** — requires/invites Holy Spirit agency, or replicable by secular practitioner?
8. **Priority Verdict** — High / Medium / Low with one-sentence reason

## Tool 1 — Post-Session Integration Guide

**File:** [[Covenant Identity — Post-Session Integration Guide]]
**Purpose:** Client-facing guide for the 24-48 hours after a heavy session — the active formation window between session close and the next day's noise.
**Priority verdict:** High. The Spirit continues what was opened in session; clients without guidance either re-armor immediately (Warrior), shut down (Hermit), or manage the content away (False Noble).

### Design Decisions
- **Three source pillars:** Thompson's implicit memory framework (the 24-48 hour processing window is neurologically active), Allender's wound-type response patterns (each type destabilizes differently after heavy sessions), and biblical post-encounter precedents (the theological frame).
- **Biblical precedents as the governing frame** — not clinical coping language. Four precedents, each contributing a specific principle:
  - Elijah (1 Kgs 19) → body care before revelation; the journey is sometimes too great
  - Moses (Exod 34) → others see what happened before you do; don't force assessment
  - Jacob (Gen 32) → the limp is proof of contact, not failure
  - Transfiguration (Matt 17) → the touch before the rising; reception precedes action
- **Practices ordered by weight** — body care first, then receptive prayer, then Scripture anchor, then brief naming, then one safe person. The Warrior will skip to cognitive work; the Hermit will skip to isolation; the False Noble will skip to resolution. The order is the formation.
- **"What to hold lightly" section** — names the specific mismanagement moves each type makes and gives explicit permission not to perform normalcy.

### Key Distinctives
Spirit-receptivity is the posture, not coping. The closing anchor (Phil 1:6) frames the 24-48 hours as the Spirit's ongoing work, not the client's management task. Cannot be replicated by secular practitioners because the frame is encounter with the God who opened what was opened.

### Connects To
- [[Covenant Identity — Character Wound Between-Session Practices]] — wound-type response patterns in the integration guide connect directly to the full wound-type practices
- [[Covenant Identity — Lament Practice Guide]] — if the session involved lament work, the integration guide is the prior step before lament is re-entered
- [[Covenant Identity — Phase-Specific Reflection Guide]] — the brief naming practice in the integration guide is the simpler precursor to between-session journaling

## Tool 2 — Character Wound Between-Session Practices

**File:** [[Covenant Identity — Character Wound Between-Session Practices]]
**Purpose:** Phase-variant, wound-type-specific between-session practices for Warrior, Hermit, and False Noble across Phases 2, 3, and 4.
**Priority verdict:** High. The Character Wound Diagnostic Tool does in-session diagnostic work — this is its direct between-session extension. The model's diagnostic precision is wasted if clients go home with undifferentiated practices.

### Design Decisions
- **One document with phase-variant sections per wound type** (not separate documents per type or per phase) — allows coach to navigate and extract the relevant section without cross-referencing multiple files.
- **Wound/God-rep intersection precondition table** — six high-risk combinations with required preconditions before assignment. These intersections are not in the Between-Session Support Materials framework (which is organized by God-rep + phase, not wound + God-rep). New content.
- **Spirit-dependency anchor per wound type** as a discrete named section — not a generic "invite the Spirit" appended at the end. Each type's section names the Spirit's specific work for that wound:
  - Warrior: making it safe to be held
  - Hermit: making it safe to be seen
  - False Noble: making receiving possible
- **Behavioral practices included** — received help (Warrior), unhiding practice (Hermit), striving interrupt (False Noble). Schema change requires behavioral data; cognitive reframe alone does not shift implicit memory.
- **Warrior: journaling contraindicated in Phase 2** — becomes intellectual management of the wound. Named explicitly rather than left to coach intuition.
- **Hermit: solo encounter practices contraindicated** until God-rep stabilizes as gentle and safe. Community bookend is non-negotiable.
- **False Noble: anti-performance interrupt** built into client-facing language at every phase — "this is not something to get right."

### Key Distinctives
Co-optation risk is named per practice per wound type within the practitioner notes. The wound pattern co-opts the practice before it can reach the wound — naming this per type is what distinguishes this from generic wound-type content. The Spirit's specific transforming work per wound type is architecturally necessary: remove the Spirit and you have a well-differentiated coping guide.

### Connects To
- [[Covenant Identity — Character Wound Diagnostic Tool]] — the in-session diagnostic that generates the wound type assigned here
- [[Covenant Identity — God-Representation Between-Session Practices]] — wound/God-rep intersection preconditions cross-reference
- [[Covenant Identity — Post-Session Integration Guide]] — after Phase 2 wound-mapping sessions

## Tool 3 — Lament Practice Guide

**File:** [[Covenant Identity — Lament Practice Guide]]
**Purpose:** Practitioner reference and client-facing guide for between-session lament practice — from Phase 2 entry through Phase 4 lifelong rhythm.
**Priority verdict:** High. Lament-to-trust is one of five named intervention modalities; introduced in Phase 2 session tools. No between-session guide existed for how to sustain the practice outside sessions.

### Design Decisions
- **Theological container is load-bearing** — must be given in session before the guide is assigned. Many clients carry the implicit conviction that honest complaint to God is faithlessness. Without the explicit frame, lament produces shame, not formation. Content: lament as God-given language (one-third of the Psalms), Jesus's lament from the cross (Matt 27:46), lament vs. despair (direction of address is the distinguishing mark), Brueggemann's orientation/disorientation/reorientation.
- **Biblical lament structure as inhabitable shape, not formula** — four movements (address, complaint, petition, trust statement) offered as a form to enter, not steps to complete in order. Psalm 88 named explicitly: lament that ends in darkness ("darkness is my closest friend") is canonical and valid.
- **Permission to lament as a discrete section** — especially for Hermit ("you will not overwhelm God with what you feel") and False Noble ("you do not need to have earned the right to lament").
- **Psalm selection calibrated by wound type AND phase** — not one master list. Phase 2 = psalms that hold lament without forced resolution (Ps 88, 13:1-4, 73:1-16). Phase 3 = psalms that move through lament toward trust (Ps 22 full, 42, 77). Phase 4 = communal lament psalms (Ps 44, 79, 80) and Psalter rotation.
- **Wound-type failure modes table** for in-session use — what each type does wrong with lament and the coach's specific response.

### Key Distinctives
Categorically Spirit-dependent: lament is addressed to God, making it structurally irreplicable by secular practitioners. Romans 8:26 (Spirit intercedes with groanings) is active in lament. The Psalms are Spirit-breathed (2 Tim 3:16); praying them is returning God-given words to God. A secular therapist can assign expressive writing; they cannot assign lament.

### Connects To
- [[Covenant Identity — Phase 2 Session Tools (Wound Mapping)]] — lament introduced there; this guide is the between-session extension
- [[Five Intervention Modalities — Definitions & Deep Elaboration]] — lament-to-trust as named modality
- [[Covenant Identity — Phase-Specific Reflection Guide]] — Phase 2 reflection and lament are companion practices

## Tool 4 — God-Representation Between-Session Practices

**File:** [[Covenant Identity — God-Representation Between-Session Practices]]
**Purpose:** Phase-variant between-session practices for five God-rep types; Gospel Encounter Library as the primary Phase 3 formation mechanism.
**Priority verdict:** High. God-Rep Deepening Tool does in-session work. Implicit God-image shifts through repeated relational experience — which happens between sessions, not in them.

### Design Decisions
- **Five God-rep types** — Punitive, Absent/Distant, Permissive/Safe, Transactional, and Overwhelming/Consuming. The fifth type (Overwhelming/Consuming) was added because the Hermit's core God-rep distortion was not cleanly represented in the original four-type framework. Flagged for update to [[Covenant Identity — God-Representation — Research Basis & Synthesis]].
- **Explicit/implicit God-image distinction as Section I** — client-facing frame explaining why their stated theology and functional experience of God can differ. Without this, clients don't understand why the practices are needed ("I already know God is good").
- **Gospel Encounter as the primary Phase 3 mechanism** — relational encounter with who Jesus actually was toward people, not theological information about God. How to read: slow, once through, two questions (where do you find yourself; what does Jesus's response feel like directed at you), no application extracted.
- **Gospel Encounter Library organized by God-rep type** — each passage cross-referenced to the specific distortion it addresses and why.
- **Phase 1 = noticing only** — no transformation practices. Phase 3 = primary transformation. Phase 4 = sustainability.
- **Practitioner failure-mode table** — what each God-rep type does in practice and the in-session response.

### Key Distinctives
Spirit-dependency is architectural: the implicit God-image shifts through encounter with the real relational Person, not a therapeutic construct or a projection. The secular parallel (attachment repair through relational experience) is real but incomplete — it treats God as an attachment figure, not as the God who is actually there. The Gospel Encounter practice works because the Spirit applies the text personally, not because the imagination is well-used.

## Tool 5 — Scripture Saturation & Meditation Guide

**File:** [[Covenant Identity — Scripture Saturation & Meditation Guide]]
**Purpose:** Teaches clients how to engage Scripture formatively (not informationally). Activation layer for Scripture anchors already embedded in existing tools.
**Priority verdict:** High. "Beholding produces becoming" (2 Cor 3:18) is the named formation mechanism throughout the model. Every major tool contains Scripture anchors. No guide taught clients how to engage those anchors below the cognitive layer.

### Design Decisions
- **Hagah (הָגָה) as the theological anchor** — the Hebrew word for meditation (Ps 1:2, Josh 1:8): to mull, to mutter, to ruminate. Returns to the same text repeatedly rather than extracting and moving on. This framing distinguishes the practice from Bible study before the client opens a text.
- **Five-approach menu** — not a single method. Different wound types and phases suit different approaches: slow reading (lectio divina), single-verse rumination (hagah), personalization (I-statement), embodied repetition (spoken + written), Gospel imagination. Coach assigns; client doesn't choose.
- **Text assignment stays with the coach** — the guide teaches the how; the coach supplies the what from existing tool libraries. Section VII cross-references where calibrated texts live (Lie Eliminator, Identity Installer, Character Wound Practices, New Covenant Promise Map, Lament Guide, God-Rep Practices).
- **Wound-type approach calibration table** — which approaches suit which types and the specific cautions per type (Warrior → analysis risk, Hermit → overwhelm risk, False Noble → performance risk).
- **Gospel imagination: Phase 3 only** — contraindicated for Overwhelming/Consuming God-rep type until encounter safety is established. Not appropriate for Phase 1 or 2.

### Key Distinctives
Three layers of Spirit-dependency unavailable to secular analogues: (1) Spirit authored the text (2 Tim 3:16); (2) same Spirit illuminates it in the reader (1 Cor 2:10-12); (3) Word is living and active — does formative work human words cannot (Heb 4:12). A secular therapist can assign slow reflective reading. They cannot assign Scripture engagement where the Spirit-inspired text works at the pre-cognitive level.

### Connects To
- [[Covenant Identity — God-Representation Between-Session Practices]] — Gospel imagination approach shared; Gospel Encounter Library cross-referenced
- All tools with Scripture anchors (Lie Eliminator, Identity Installer, Character Wound Practices, New Covenant Promise Map, Lament Guide)

## Tool 6 — Phase-Specific Reflection Guide

**File:** [[Covenant Identity — Phase-Specific Reflection Guide]]
**Purpose:** Noticing-oriented journaling prompts calibrated to each phase; between-session interior tracking; preparation for the next session.
**Priority verdict:** Medium-high. Most accessible tool in the library; lowest readiness floor. Extends session processing. Lowest Spirit-dependency — the tool that could most easily become secular if the Spirit-invitation is removed.

### Design Decisions
- **Noticing vs. processing as the load-bearing frame** — must be given in session. Processing asks "what does this mean?" Noticing asks "what is present?" The distinction determines whether the tool engages implicit content or stays cognitive.
- **Spirit-invitation as required entry posture** — one sentence before opening the journal: *"Holy Spirit, surface what You want me to notice. I'm listening."* This is what distinguishes the practice from secular journaling. It is not optional.
- **3-4 prompts maximum per phase** — more produces overwhelm for Hermit and performance for False Noble.
- **Anti-performance language built in at every phase** — especially critical for False Noble: "Do not write what a person who is growing should write. Write what is actually true."
- **Do not require sharing** — journaling-as-homework produces performance. The journal is the client's own record.
- **Time limits specified per phase** — Phase 2: 10 minutes maximum (longer risks processing beyond what the session opened). Phase 4: 20-30 minutes (Growth Report depends on richness of what surfaces).
- **Phase 4 prompts designed to surface Growth Report language** — verbatim phrases from Phase 4 entries are the material the Growth Report is built from.
- **God-rep flags in practitioner notes** for specific prompts that activate God-rep distortions (e.g., "what do you sense God thinks?" will activate punitive image).

### Key Distinctives
Spirit-invitation is the thin but real distinction from secular journaling. The prompts are calibrated to a Spirit-dependent formation model — they track what the Spirit is doing in the client's formation, not generic self-reflection. Weakest Spirit-dependency in the library; requires the Spirit-invitation to remain distinctively Christian.

## Tool 7 — Spiritual Disciplines Guide

**File:** [[Covenant Identity — Spiritual Disciplines Guide]]
**Purpose:** Willard VIM-grounded discipline library calibrated to wound type and phase; co-optation signal tables; rule-of-life construction for Phase 4.
**Priority verdict:** High. Built 2026-05-20. Disciplines are the Means layer of VIM. Without this, formation work has no sustained between-session vehicle covering the whole person. Every other between-session tool is essentially a discipline or variant of one. *Built 2026-05-21: Between-Session Distress Protocol — see Tool 8.*

### Design Decisions
- **Willard's abstinence/engagement taxonomy as organizing framework** — already in Practitioner Theory library (Spiritual Formation.md). Gives the tool theological backbone without constructing a new taxonomy. 13 disciplines total: 6 abstinence (silence, solitude, fasting, secrecy, submission, frugality), 7 engagement (study, worship, prayer, fellowship, confession, service, celebration).
- **Co-optation signal table is the highest-value content** — for each wound type, two columns: formative signal vs. co-optation signal per discipline. This content does not exist anywhere else in the system. The wound pattern co-opts the discipline before it can reach the wound; naming this per discipline per type is what makes this tool clinically distinct from generic spiritual disciplines content.
- **Phase sequencing grid** — wound type × phase matrix for deployment decisions. Coach assigns 1-2 disciplines at a time, never a comprehensive list.
- **Training vs. trying distinction in client frame** — prevents disciplines from becoming law. "You are not trying harder to be holy. You are training your whole person to receive what the Spirit offers."
- **Fasting caution across all types in Phase 2** — added physiological stress during emotionally demanding wound work; stated explicitly in the phase grid.
- **Hermit: solitude must have community bookend** — never assign solitude alone to a Hermit. The co-optation warning is prominent.
- **False Noble: "everything going well" is the suspicious signal** — the False Noble co-opts nearly any discipline by turning it into performance. Consistent, enthusiastic engagement is the flag, not failure.
- **Rule-of-life construction as Phase 4 closing deliverable** — structured process: which disciplines moved something, which was most resisted, co-optation risk for each chosen discipline, form/frequency/accountability. One-page document owned by the client; referenced in Growth Report.

### Key Distinctives
Spirit-dependency is architectural — Willard's own claim: disciplines do not produce transformation; the Spirit produces transformation through the conditions the disciplines create. Remove the Spirit and the disciplines become virtue cultivation. With the Spirit, the Warrior's submission practice trains the body to know what trust feels like. No secular practitioner can assign submission with the same mechanism because the formation target is trust in a real God, not behavioral compliance.

### Connects To
- [[Covenant Identity — Character Wound Between-Session Practices]] — wound-type formation needs inform which disciplines are assigned
- [[Covenant Identity — God-Representation Between-Session Practices]] — disciplines that directly activate God-rep flagged per type
- [[Covenant Identity — Phase-Specific Reflection Guide]] — reflection accompanies discipline practice; the client brings what happened in the discipline to the journal
- [[Covenant Identity — Growth Report Template]] — rule of life built in Phase 4 is referenced in the Growth Report

## Tool 8 — Between-Session Distress Protocol

**File:** [[Covenant Identity — Between-Session Distress Protocol]]
**Purpose:** Care document defining what qualifies as between-session distress, giving the client a step-by-step response sequence, and giving the coach a response protocol (acknowledgment window, bridge call structure, escalation decision tree). Embedded client card for direct handout.
**Priority verdict:** High. Built 2026-05-21. This is a care responsibility, not a service enhancement — Phase 2 work opens wound material that can surface acutely between sessions. A client without a protocol is left to manage something the coaching activated without any framework for what to do.

### Design Decisions
- **Three-tier distinction as the governing frame** — Tier 1 (Normal Processing: 24-48hr, self-settling), Tier 2 (Between-Session Distress: elevated, not crisis), Tier 3 (Crisis: escalate immediately). Client and coach need shared language before contact occurs; without it, clients either over-contact or under-contact.
- **Practitioner reference with embedded client card** — same document structure as other library tools. Practitioner reads Sections 1–5 before assigning. Client receives the CLIENT CARD section only.
- **Client card ordered by triage, not by instruction weight** — Step 1 is always distress/danger differentiation. If danger, go to crisis section. If distress, body first, then coach contact. The order protects against the client skipping to coach contact before self-stabilizing.
- **Wound-type calibration within the client card** — three short paragraphs, one per type, naming the specific pattern co-optation for that type: Warrior (push through / armor up), Hermit (go silent / manage alone), False Noble (perform okayness / turn distress into productivity). These are the most likely ways each type fails to use the protocol.
- **Coach response protocol with explicit placeholders** — acknowledgment window, contact method, and response form are coach-specified fields. These must be filled in before the card is given.
- **Bridge call structure is containment, not coaching** — five-step call structure, explicitly prohibiting excavation or opening new material. The bridge call's only job is to stabilize in the present and confirm tier level.
- **Frequency-of-use as a diagnostic signal** — if the protocol is used two or more times in a month, the engagement is moving faster than the client's window of tolerance or the material is clinical in depth. Do not optimize the protocol — read what the frequency is saying.

### Key Distinctives
Theologically grounded in covenant presence: Step 2 speaks truth *to* God ("God, you see me right now") not self-affirmation into the air. The four Scripture passages in Step 4 are not crisis-management texts — they are covenant-presence anchors that locate the client in God's address. The wound-type notes are the only between-session calibration for this specific activation pattern — they address the failure mode before it occurs.

- [[Covenant Identity — Post-Session Integration Guide]] — adjacent tool; Integration Guide covers the 24-48 hours after a heavy session; Distress Protocol activates when the Integration Guide practices are not sufficient
