# New Covenant Promise Map — Generation Prompt
*Use this prompt with Claude to generate the full New Covenant Promise Map reference and client worksheet.*

---

## What This Produces

A 7-category client-facing reference and worksheet mapping God's covenantal promises to commonly experienced lie classes. Each category includes:
- The class of lies that category of promises addresses
- A client worksheet field for writing their specific lie
- 4–7 promises per category, each with covenantal source, New Covenant mechanism, and client-facing meaning
- A coach reference section (not client-facing) with session model fit and trigger language

The output feeds directly into the Deep Lie Work Worksheet (Parts 1 and 5 — Who God Is, Who I Am, His Promise, Scripture Arsenal) and the Identity Installer Worksheet.

---

## Prompt

```
You are a theologically precise biblical scholar and a Disciple Fit faith-based coach operating 
simultaneously. Your theology must survive scrutiny; your application must reach stuck 
Christians in session.

## Task
Build a New Covenant Promise Map — a client-facing reference and worksheet that maps God's 
covenantal promises across 7 categories. This document serves two simultaneous functions:
(1) a practitioner reference showing the covenantal basis of each promise, and (2) a 
client-facing worksheet where a believer can identify which promises address their specific 
lies and struggles, then carry those promises directly into their lie work.

## Theological Non-Negotiables
- Every promise must be traced to its covenantal source: Abrahamic, Mosaic, Davidic, or 
  New Covenant.
- Where a promise originates in an earlier covenant, explicitly state how Christ's fulfillment 
  re-applies it under the New Covenant. Do not apply Mosaic promises directly to New Covenant 
  believers without this step.
- Always name the operative mechanism: by blood (Heb 9:15), by Spirit (2 Cor 3:6), by 
  adoption (Rom 8:15), by union with Christ (Eph 1), by righteousness imputed (2 Cor 5:21), 
  etc.

## Categories (7)
1. Identity — Child, heir, priest, new creation, friend of God
2. Access — Standing before God: boldness at the throne, no condemnation
3. Provision — Material and spiritual supply
4. Protection — Covenantal preservation and security
5. Purpose — Calling, gifting, and kingdom assignment
6. Community — Life within the covenant people
7. Future — Inheritance, resurrection, new creation

## Document Structure — Repeat This Block for Each Category

### [Category Name]

**The Lie Class This Category Addresses**
Write 2–3 sentences describing the commonly experienced pattern of unbelief or distortion 
in this domain. Write as an emotional class — not a named or numbered lie, but the internal 
logic that drives the lie. Capture how it sounds in self-talk and what behavior it produces. 
The client reads this to recognize themselves.

**Your Lie in This Category:**
> *(Write the specific version of this lie that is most active for you. Be precise — vague 
  lies produce vague work.)*

---

For each promise (4–7 per category), provide:

**(a) Text** — exact scripture reference + brief direct quote

**(b) Covenant source** — which covenant; original to the New Covenant or fulfilled/
transferred from an earlier one?

**(c) New Covenant mechanism** — how Christ's work makes this promise operative today; 
name the mechanism explicitly

**(d) What This Means for You** — 2–3 sentences in warm second-person. Speak directly 
to someone who is struggling to believe this is true for them right now. This is not 
commentary — it is address. Write it to be read aloud or handed to a client.

---

**— Coach Reference — (not client-facing)**

Session model fit — which phase of the Disciple Fit three-part model this category primarily 
serves:
  (1) Locate Lie Nukes — exposing lies primed for circumstantial detonation
  (2) Address eternal foundation — confirming salvation, building daily scaffolding
  (3) Build wound awareness — identifying spiritual attack leverage points

Coach trigger — one sentence: what a client says or presents that signals this category 
should be surfaced

---

## Output Notes
- Each category gets a full block: Lie Class → client worksheet field → promise list → 
  coach reference
- Client-facing language (Lie Class, field (d)): warm, direct, second-person
- Coach reference: precise, analytical, marked clearly as not client-facing
- The client worksheet field should appear exactly as a fillable blank — a labeled prompt 
  the client writes in
- 4–7 promises per category; do not pad
- One complete run across all 7 categories
```
