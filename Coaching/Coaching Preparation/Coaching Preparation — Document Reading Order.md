# Coaching Preparation — Recommended Reading Order

The governing principle: theological identity before psychological mechanism before methodology before tools. The biblical framework determines which psychological mechanisms are valid; mechanism determines which methods are appropriate; methods determine which tools to reach for. If you skip the foundation blocks and go straight to methods, you will know how to run the tools but not why or when.
