# Relational Performance Advisory — Multi-Rater 360 Instrument

**What this is:** Complete 360 assessment instrument for the RPA engagement. Three sections: (1) Rater Survey — client-facing, administered to supervisor/peers/direct reports; (2) Self-Rating Survey — administered to the leader being assessed; (3) Practitioner Guide — administration, scoring, interpretation, and integration into the Behavioral Pattern Profile. Andrew-only.

**Abbreviated version (Month 12 checkpoint):** 15 core items are marked with ★. Use only these items for the abbreviated re-assessment. Same five clusters, 3 items each, same open-text prompts.

**Format note:** Sections 1 and 2 are designed to be copied into a digital survey tool (Google Forms, Typeform, etc.) with [LEADER NAME] replaced by the leader's first name before deployment.

# SECTION 1 — RATER SURVEY

*Client-facing. Deploy to: supervisor, peers, direct reports. Minimum 3 raters per category.*

---

## Introduction

You have been asked to complete a brief behavioral survey about **[LEADER NAME]** as part of a leadership development process. This is not a performance review. It is a structured observation — an opportunity to share what you have directly seen in how [LEADER NAME] works with people.

**Confidentiality:** Your responses are confidential. [LEADER NAME] will receive a summary of overall themes and patterns across all raters — not individual responses that can be traced back to you. Andrew Houser will see the raw data; no one else will.

**How to answer:** Rate only what you have directly observed. If you have not had the opportunity to observe a particular behavior, select "No opportunity to observe" — do not guess. There are no right or wrong answers; what is useful is what is accurate.

**Time:** Approximately 15–20 minutes.

# SECTION 3 — PRACTITIONER GUIDE

*Andrew only. Not shared with leaders or sponsor.*

---

## Administration Protocol

**Timing:** Deploy after all narrative interviews are complete. Do not administer 360 before narrative interviews — survey language primes how leaders talk about their own behavior.

**Rater identification:** Work with the leader (not the sponsor) to identify raters:
- 1 supervisor
- Minimum 3 peers
- Minimum 3 direct reports

If the leader raises concerns about a specific rater, discuss the concern — but do not exclude raters simply because they may provide critical data. The 360 is a behavioral mirror, not a satisfaction survey.

**Rater confidentiality statement (deliver before distributing the survey):**
> "Your responses are confidential. [Leader name] will see aggregate themes and patterns from all raters combined — not individual responses attributed to you specifically."

**Collection window:** 4–6 weeks. Send one reminder at the 2-week mark. Do not send more than one reminder — pursuit behavior from the consultant models what we're trying to change in the leaders.

**Self-rating:** Administer to the leader simultaneously with rater deployment. Do not administer before narrative interview.

**Do not administer verbally.** Written/digital only. Verbal administration introduces interviewer influence and loses the rater's private reflection process.

---

## Scoring Protocol

### Step 1 — Data Preparation

For each leader, create a scoring table:

| Cluster | Supervisor Mean | Peer Mean | DR Mean | All-Rater Mean | Self-Rating | Divergence |
|---|---|---|---|---|---|---|
| Accountability Delivery | | | | | | |
| Regulation Under Pressure | | | | | | |
| Feedback Receptivity | | | | | | |
| Relational Repair | | | | | | |
| Authority Clarity | | | | | | |

- Exclude N/O responses from mean calculation (they are not zeros)
- Calculate means to one decimal place
- All-Rater Mean = mean of all rater responses combined (supervisor + peers + direct reports)
- Divergence = Self-Rating minus All-Rater Mean (positive = self-rated higher; negative = self-rated lower)

### Step 2 — Self-Rating Divergence Flag

Flag any cluster where the absolute divergence is **≥ 2.0 points**.

**Direction matters:**

| Direction                                     | What it signals                                                                                                                                                                                          |
| --------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Self higher than raters (positive divergence) | Limited self-awareness about relational impact in this domain. Leader believes they are functioning better than others observe. Handle with care in profile debrief — carries significant Status threat. |
| Self lower than raters (negative divergence)  | Underestimation. Leader is harder on themselves than the data warrants. Less common; may indicate a self-critical pattern, depression, or protective self-deprecation.                                   |

Note: mild divergence (0.5–1.5 points) is normal and expected. Divergence becomes diagnostic at ≥ 2.0.

### Step 3 — Rater Group Comparison

Compare means across rater groups. Significant variation between groups is itself diagnostic:

| Pattern | What it suggests |
|---|---|
| Direct reports rate significantly lower than peers/supervisor | Pattern activates specifically in downward authority relationships — with people the leader has power over. Most common pattern in hospitality management. |
| Supervisor rates significantly lower than peers/direct reports | Pattern activates upward — with people the leader reports to. Suggests the supervision relationship is where disorganization or regulation breakdown appears. |
| Peer ratings significantly diverge from direct report ratings | Leader manages up and across well but struggles in the accountability role. |
| All groups rate similarly | Pattern (or collaborative capacity) is consistent across relationship types — the most informative finding in either direction. |

---

## Interpretation — Cluster to Pattern Mapping

Cluster scores do not map 1:1 to behavioral patterns. Pattern identification requires triangulation with narrative interview and facilitated observation data. However, each cluster has primary diagnostic weight toward specific patterns:

| Cluster | Low Score (1–2.5) Most Consistent With | High Score (4–5) Most Consistent With |
|---|---|---|
| **Accountability Delivery** | Withdrawal (avoids the conversation) OR Disorganization (inconsistent follow-through) | Collaborative capacity OR Escalation (if only low-stakes items score high) |
| **Regulation Under Pressure** | Escalation (flooding, pursuit, reactivity) OR Withdrawal (shutdown, avoidance) | Collaborative capacity — the most diagnostically significant high score |
| **Feedback Receptivity** | Escalation (defensiveness, counter-attack) OR Disorganization (inconsistent — sometimes open, sometimes closed) | Collaborative capacity |
| **Relational Repair** | Withdrawal (avoids re-entry after rupture) OR Escalation (pursues resolution aggressively, repair attempts feel threatening) | Collaborative capacity |
| **Authority Clarity** | Disorganization (role boundary confusion, inconsistent authority, compliance/resistance cycling) | Collaborative capacity OR Escalation (if authority is exercised through intensity rather than clarity) |

### Pattern Signature Profiles

**Withdrawal pattern:**
Accountability Delivery: Low | Regulation Under Pressure: Low-to-Moderate (appears composed, but through avoidance) | Relational Repair: Low | Authority Clarity: Low-to-Moderate (defers or hedges)
Narrative interview confirmation: avoidance language, passive framing, exits relational field in accounts

**Escalation pattern:**
Accountability Delivery: Variable (high on initiation, low on sustained/regulated delivery) | Regulation Under Pressure: Low | Feedback Receptivity: Low | Relational Repair: Variable (initiates repair but may overwhelm)
Narrative interview confirmation: high affect in conflict accounts, personalizes others' behavior, pursues in accounts

**Disorganization pattern:**
Authority Clarity: Low | Accountability Delivery: Variable and inconsistent | Regulation Under Pressure: Variable (not predictably high or low)
Narrative interview confirmation: contradictory descriptions of authority figures, compliance/resistance cycling

**Collaborative capacity:**
All clusters 4+, OR consistent scores with isolated lower cluster that becomes the coaching target

## Open-Text Synthesis

Open-text responses are not coded quantitatively. Use them for:

1. **Behavioral specificity** — concrete examples that confirm or complicate the quantitative profile
2. **Language patterns** — note the language raters use to describe the leader; vivid or emotionally charged language from raters on a specific cluster is itself a signal
3. **Silence** — clusters that produce brief, vague, or absent open-text responses despite sufficient rater tenure indicate either low behavioral visibility (the behavior doesn't surface much) or rater avoidance (they observed something they don't want to put in writing)
4. **Cross-rater consistency** — if multiple raters independently describe the same specific behavior, it is high-confidence data regardless of quantitative score

**Do not share raw open-text responses with the leader.** Synthesize themes. The aggregate framing is: "Raters consistently described..." or "Several raters noted..."

---

## Integration into the Behavioral Pattern Profile

360 data feeds Sections 1, 2, and 4 of the Behavioral Pattern Profile:

**Section 1 — Behavioral Pattern Summary**
- Cluster means establish the four-dimension ratings (Low / Moderate / High presence per dimension)
- Language model: "360 data from [N] raters across supervisor, peer, and direct report categories showed [specific cluster finding]. Direct report ratings on [cluster] were notably [lower/higher] than peer ratings, suggesting the pattern activates most consistently in [downward/upward] authority relationships."

**Section 2 — Activation Profile**
- Rater group divergence identifies which relationship types activate the pattern
- Self-rating divergence informs self-awareness rating and coaching entry point

**Section 4 — Source Notes (internal — not included in leader's copy)**
- Self-rating scores with divergence flags
- Raw rater group means
- Any open-text observations that will not appear in the profile but informed interpretation
- Note which rater group had the most internally consistent vs. divergent responses

## Abbreviated 360 — Month 12 Protocol

Use ★-marked items only (3 per cluster = 15 items total). Same open-text prompts. Same rater pool where possible — if raters have turned over, recruit from the same role categories.

Do not introduce new items at Month 12. The measurement value is in the comparison to baseline, which requires the same items.

Scoring protocol is identical. Report cluster-level movement (mean change from baseline to Month 12) at the aggregate level in the Month 12 sponsor report.
