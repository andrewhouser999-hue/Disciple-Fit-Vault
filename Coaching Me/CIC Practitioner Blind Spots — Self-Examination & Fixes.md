# CIC Practitioner Blind Spots — Self-Examination & Fixes

**What this is:** Five structural vulnerabilities in how Covenant Identity Coaching could go wrong in live practice, generically — not tied to any specific client — each paired with a concrete fix. Written as a personal accountability reference, not a session tool.

**When used:** Periodic self-review before or during active CIC coaching seasons; also worth re-reading whenever a session felt "off" and the cause isn't obvious.

**Who uses it:** Andrew, as practitioner self-examination — not client-facing.

---

## 1. The scope boundary isn't decided once — it's a live call every session

**Correction (2026-07-04):** the original framing of this point assumed the Diagnostic Lens Transition Logic routes to CBT-informed tools under one ambiguous "CBT (if needed)" label, bundling coach-safe technique with clinical/referral territory. That assumption was wrong — checked against the live document, it already carries the split: **Stage 4a** ("CBT-Informed Intervention") lists specific coach-safe tools (Socratic questioning, laddering/downward arrow, thought record, behavioral experiment), triggers only on active cognitive distortion, and has an explicit return signal back to Stage 3 — "CBT tools serve the covenant work; they do not replace it." A wholly separate **"Referral Signal (exits the sequence entirely)"** section instructs: *"Do not attempt to work the distortion with CBT tools when clinical presentation is indicated"* — routing to a dedicated `Covenant Identity — Crisis & Referral Protocol.md`. Stage 4b carries its own inline referral trigger for trauma/dissociative presentation as well. The document already does the structural work this section originally called for.

What's left is not a documentary gap — it's an operational one. The document can define the exit; only real relationships and real conversations execute it. That threshold still has to be judged live, every session, by the practitioner in the room, no matter how well the logic is written down. And several of CIC's own Tier 1 sources (attachment formation, Beck's terror management, trauma-adjacent neuroscience) sit at the edge of clinical territory by design, so some version of "is this still coaching" is present in every CIC engagement even with the safeguard well-built.

**Fix (operational, not documentary):**
- Identify 2–3 licensed therapists/counselors by name before taking live clients — not a mental note, an actual referral list, ready to hand to the Crisis & Referral Protocol's existing triggers rather than inventing a new checklist.
- Confirm the Crisis & Referral Protocol is somewhere you'd actually reach for it mid-session, not just filed in Practitioner Reference.
- Name coaching's scope out loud at intake, every time, as informed consent — not just tracked internally.
- Establish a standing case-consultation relationship with someone clinically trained, even monthly. The check on judgment needs to be a person, not a document — the document already tells you when to look for one; a standing relationship makes sure one is actually there.

## 2. Mistaking the client's grasp of the framework for the Spirit doing the work

CIC's telos is the interior life — sanctification, not comprehension. A client who can now name their false identity category or locate their affections correctly will feel like progress happened. Cognitive fluency with the categories is cheap to produce and easy to mistake for formation, especially because the content is genuinely rich.

**Fix:**
- Stop crediting insight as progress in the same session it appears.
- Before introducing new material, require an answer to "what did you actually do differently since last session" — if the answer is only "I understood something," that's not yet grounds to advance.
- Let the Comprehensive Discipleship Markers function as behavioral gates, not as vocabulary the client has become fluent in.

## 3. Forcing the model's sequence onto a soul that isn't moving in that order

Affections → False Identity → Covenant Truth → CBT is a map of terrain, not the client's actual timeline. Real resistance surfaces on its own schedule, not the practitioner reference's.

**Fix:**
- Before advancing a session to the next stage, ask yourself — not the client — "is this because they're ready, or because it's next in the document."
- Give explicit permission to repeat or stall on a stage across multiple sessions without treating that as failure to progress.
- After each session, note in one line whether the move made was evidence-driven or schedule-driven. A one-line self-audit, not a new framework.

## 4. Your own unexamined patterns leaking into the diagnosis you give others

CIC diagnoses affections and false identity in clients while Andrew's own identity work (Lie Eliminator, Identity Installer, Secured in Father) remains, by design, an active design lab rather than settled ground. Whatever is still live personally has a direct route into how patterns get read in someone else — over-naming what you're primed to see, or missing it in yourself while it's operating in the room in real time.

**Fix:**
- Get an actual outside coach, counselor, or spiritual director — a real person, not another personal document. A long-running design lab is itself the evidence that self-administered work has a ceiling here.
- Run a short personal check before client sessions: am I needing this session to go well. Name it before it can quietly shape the read given to someone else.

## 5. Running the framework instead of being present

The neuroscience CIC actually cites — Perry, Siegel, Rock — says regulation and secure attachment come from attunement, not technique accuracy. A rich toolkit makes it tempting, in a stuck or sideways-going session, to reach for a documented move rather than stay a regulated, present human the client's nervous system can borrow calm from.

**Fix:**
- Default response to a stuck moment: silence and naming what's sensed, before reaching for a tool.
- Practice personal nervous-system regulation directly — grounding, breath, whatever would be prescribed to a client — not just know the literature behind it.
- Give explicit permission to say "let's sit with this" instead of deploying a script when a session goes sideways.

---

**Overall priority if only one gets acted on first:** #1. It is the only one where the cost of drift is borne primarily by someone else rather than by the coaching relationship's effectiveness.
